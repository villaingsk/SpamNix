SpamNix helps reduce comment spam with lightweight detection layers.

== Description ==
SpamNix is a lightweight anti-spam plugin for WordPress comment forms.
It adds multiple checks to detect automated submissions and suspicious content.

Core features:
- Honeypot field check
- Time trap validation
- Link and keyword scanner
- IP rate limiting
- Disposable email domain filtering

Planned PRO features:
- Advanced spam statistics dashboard
- Automatic IP blocking intelligence
- Country-based filtering
- Detailed log viewer
- Cloudflare API integration for automatic IP blocking

== Installation ==
1. Upload the `spamnix` folder to `/wp-content/plugins/`.
2. Activate the plugin from the WordPress Plugins screen.
3. Open `Settings > SpamNix` to configure each protection service.
